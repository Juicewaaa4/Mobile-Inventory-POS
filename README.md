# Mobile Inventory & POS 📦

A comprehensive Android Point-of-Sale (POS) and Inventory Management application. Built with Kotlin and Jetpack Compose, this system provides an all-in-one solution for managing sales, automated profit calculations, and real-time inventory updates.

## 🌟 Key Features

**Point-of-Sale (POS) System**
- **Efficient Checkout:** Fast and intuitive checkout workflow with dynamic quantity adjustments.
- **Flexible Payments:** Support for different payment options with necessary validation steps.

**Inventory Management**
- **Real-Time Stock Tracking:** Powered by Firebase, keeping all devices in sync instantly.
- **Automated Profit Calculation:** Direct calculation of "Cost Price" and "Selling Price" with automatic per-unit profit margin displayed.
- **Low Stock Alerts:** Automated warnings to keep track of essential inventory and prevent stockouts.

**Sales Analytics & Reporting**
- **Live Dashboards:** See real-time daily sales figures and profit margins as they happen.
- **Export Functionality:** Generate custom reports with one-click export to PDF and Excel formats. 

**Administration & Security**
- **Role-Based Workflows:** Secure operations separating standard staff access from administrator controls.

## 🛠️ Technology Stack
- **Languages:** Kotlin
- **UI Framework:** Jetpack Compose (Modern Android UI toolkit)
- **Backend/Database:** Firebase Authentication & Realtime Database

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Juicewaaa4/Mobile-Inventory-POS.git
   ```
2. **Open the Project:**
   Open the folder in **Android Studio**.
3. **Setup Firebase:**
   Ensure your `google-services.json` is correctly configured in the `app/` folder to connect to your Firebase instance.
4. **Build and Run:**
   Sync the Gradle environment and deploy the application to your Android device or emulator.
